# Documentation des Données

Les données brutes utilisées pour cette étude proviennent du benchmark **BBOB** de la plateforme **COCO**.

## Sources
Les archives de données sont disponibles publiquement à l'adresse suivante :
[https://coco-platform.org/testsuites/bbob/data-archive.html](https://coco-platform.org/testsuites/bbob/data-archive.html).

## Problèmes traités
L'analyse porte sur les fonctions suivantes en **dimension 10**:
* **f2** : Ellipsoïde (Unimodale).
* **f8** : Rosenbrock (Vallée courbée).
* **f10** : Ellipsoïde avec rotation (Non-séparable).
* **f15** : Rastrigin (Multi-modale).
* **f21** : Gallagher's Gaussian 10-peaks.
